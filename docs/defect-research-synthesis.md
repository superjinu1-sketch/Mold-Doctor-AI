## 0. 분류 체계 (3축 교차)

```
1차축  공정 Phase   : 재료준비 → 충전(filling) → 보압(packing) → 냉각(cooling) → 이형(ejection)
2차축  결과 유형     : 표면(surface) / 치수·형상(dimensional) / 내부·구조(internal)
원인축 4M           : Machine / Material / Mold / Method  (+ Man, 환경)
```

핵심 원칙: **대부분 불량은 단일 Phase가 아니라 복합 원인.** 트리는 "Primary Phase(1순위)"와 "Contributing(기여)"를 구분한다. ✅

Phase → 4M 주도 도메인:
- 재료준비 → Material (수분·오염·열화)
- 충전 → Machine + Mold + Method (유동 선단 거동)
- 보압 → Machine(압력) + Mold(게이트 설계)
- 냉각 → Mold(냉각채널) + Method(냉각시간)
- 이형 → Mold(발구·이젝터) + Machine

RJG Decoupled Molding 제어변수 매핑: 멜트온도→플로마크·버닝·웰드·은선 / 충전속도→제팅·에어트랩·버닝 / 보압→싱크·보이드·플래시·잔류응력 / 냉각→휨·치수·이형·크랙. ✅

## 1. 충전 단계 (Filling Phase)

### 1.1 Short Shot (미성형/숏샷)
- **식별**: 말단·리브·보스 미충전. 단면 매끄럽고 광택=단순 미성형 / 끝단 탄화=Air Trap 겸발. 복수 캐비티 균등 미충전=사출량/밸런스. ✅
- **메커니즘**: 유동 에너지(압력×온도) < 유동 저항 → 선단 동결(freeze-off), 또는 에어포켓이 진입 차단. ✅
- **4M 원인**: Machine(사출량·압력·속도 부족, 배압부족→계량불안정) / Material(점도과다=온도낮음·MFI낮음, 미건조) / Mold(게이트·런너 과소, 벽두께 과소=L/T초과, 벤트막힘, 금형온도 과저) / Method(멜트온도 낮음, 속도 과저→냉각손실).
- **우선순위**: ①사출 압력·속도+멜트온도 부족(충전 마지막 20~30% 최대저항 구간) ②게이트/런너 치수(특정캐비티만이면 밸런스) ③벽두께 L/T 초과 ⚠️ ④Air Trap. 조건전환: 특정부위 반복→Mold / 배치산발→Material·계량 / 금형온도 변경후→Method. ✅
- **검증**: Progressive Short-Shot Study(보압0, 사출량 20→90% 단계, 선단·에어트랩 위치 특정), 점도곡선(속도별 peak압×fill time→플래토), Gate Seal study. ✅
- **조정**: 멜트온도↑, 금형온도↑, 사출속도↑(과하면 제팅/플래시), 사출압↑, 배압 소폭↑, 사출량↑, 게이트확대, 벤트 추가.

### 1.2 Flash (플래시/버)
- **식별**: 파팅면 전체둘레=클램프부족·과압 / 특정위치 반복=금형마모·핀clearance / 조건변경후 급발=과압·고온 / 이젝터핀 주변=clearance. ✅
- **메커니즘**: 캐비티 압력×투영면적(separation force) > 클램프력 → 파팅면 미세개방→침투. 또는 파팅면 틈새(마모). 클램프력 = 캐비티압×투영면적×안전계수(1.1~1.2). ✅
- **4M**: Machine(클램프력 부족, 토글 마모, 과도 사출압·속도) / Material(저점도=PP·PA·POM·LCP, 고온) / Mold(파팅면 마모·이물, 금형 deflection, 평행도, 핀clearance) / Method(V/P 전환 지연→과보압, 홀드압 과고).
- **우선순위**: ①클램프력 부족+과도 사출/홀드압 ②파팅면 마모(특정위치) ③저점도수지+고온. 역설주의: **과도 클램프력→금형변형으로 벤트 압착→Diesel(버닝) 겸발.** ✅
- **검증**: 클램프력 계산검증(투영면적×캐비티압÷9.8=ton), 블루마킹(밀착패턴), V/P 전환위치 모니터.
- **조정**: 클램프력↑(과도주의), 사출압↓, 사출속도↓, 멜트온도 소폭↓, 홀드압↓, V/P 전환 앞당김, 파팅면 재연마.

### 1.3 Jetting (제팅)
- **식별**: 게이트에서 시작하는 구불구불한 지렁이/뱀 줄무늬(worm track). 동심원=플로라인, 직선=웰드(게이트서 시작 안함)와 구분. ✅
- **메커니즘**: 관성력 > 점성력 → fountain flow 대신 자유제트. 게이트 통과 유속이 임계속도 초과(사례 263mm/s, 조건의존 ⚠️), 단면 급팽창 구조. ✅
- **4M**: Machine(사출속도 과고) / Material(멜트온도 과저→상대 관성↑, 저점도) / Mold(게이트 과소·위치불량=오픈공간 정면, 벽두께 급변, 금형온도 과저→제트 고화) / Method(다단사출 미적용).
- **우선순위**: ①게이트 위치/크기+사출속도 과고 ②게이트가 벽면 충돌 없이 오픈공간 향함. 속도만 낮춰 소멸→Process / 소멸안됨→Gate설계. ✅
- **검증**: 다단사출 테스트(1단 속도만↓), 게이트 두께 ≥ 벽두께 50~80% ⚠️.
- **조정**: 게이트통과 속도↓(다단), 멜트온도↑, 금형온도↑, 게이트 확대, 위치 벽면방향, 팬/탭 게이트.

### 1.4 Weld Line / Meld Line (웰드라인)
- **식별**: 홀·보스·코어핀 주변, 멀티게이트 합류부 가는 선. 합류각<135°=weld(V홈·강도저하) / >135°=meld(강도저하 적음). GF수지=선 위치 강도 모재의 50~80%↓. ✅
- **메커니즘**: 선단 표면 냉각·배향 후 재합류 → 재확산·상호침투 불충분→계면 약화. 멜트 220→250℃시 V홈 깊이 7→3μm. GF=합류면 섬유 수직배향→강도 대폭↓. ✅
- **4M 우선순위(Taguchi 논문)**: ①**멜트온도 부족(강도 기여 70.98%)** ②홀드압(14.12%) ③사출속도(13.14%) ④사출압(0.82%). 구조적(Mold): 홀/코어 있으면 웰드 불가피→위치를 비고응력부로 이동·보조게이트로 합류각 변경이 근본대책. ✅
- **검증**: 웰드위치 강도 vs 무웰드 비교, 멜트온도 +10℃후 V홈 감소, Moldflow 합류각 예측 ⚠️.
- **조정**: 멜트온도↑(최우선), 금형온도↑, 홀드압↑, 사출속도↑, 게이트위치 이동(근본), 웰드 예측위치 벤트, 고유동 그레이드.

### 1.5 Air Trap & Burn Mark / Diesel Effect (에어트랩+버닝)
- **식별**: 흑갈~흑색 탄화, 캐비티 최말단·리브끝·보스내부·코너, 탄냄새(POM·ABS). 은선(흰색·게이트서 시작·탄화없음)·변색(전체·위치선택성 없음)과 구분. ✅
- **메커니즘**: 갇힌 공기 급속압축(단열압축, 디젤원리)→온도 급상승→자연발화→탄화. ✅
- **4M**: Machine(사출속도 과고, **클램프력 과고→벤트 압착**) / Material(ABS·POM 휘발성, 고온) / Mold(벤트 없음·막힘, 게이트 위치, 리브 깊이:두께>1.5:1) / Method(속도 과고, 다단 미적용).
- **벤트 깊이 기준**: PP·PE 0.013~0.030in(0.33~0.76mm) / ABS·PC 0.025~0.038in. 결정성 0.025mm·비결정성 0.038mm ✅. 랜드폭 3~6mm.
- **우선순위**: ①**벤트 부족/막힘**(SPE: "디젤효과는 항상 불충분 벤팅의 신호") ②사출속도 과고 ③게이트위치 ⚠️. 기존금형 갑작발생=벤트막힘(탄화퇴적) / 신금형=벤트설계 누락.
- **검증**: Progressive Short-Shot(에어트랩 위치), 벤트청소후 재시험, 속도 30~50%↓ 테스트.
- **조정**: 벤트 추가·청소(최우선), 사출속도↓, 클램프력 필요최소로↓, 멜트온도 소폭↓, 게이트위치, 리브 깊이:두께≤1.5:1.

## 2. 보압·냉각 단계 (Packing / Cooling Phase)

### 2.1 Sink Mark (싱크마크/수축자국)
- **식별**: 두꺼운 벽·리브 이면·보스 이면 얕은 함몰. **Sink vs Void**: 동일근원(국부 체적수축), 스킨 유연→안으로 끌림=Sink / 스킨 견고→내부 진공=Void. 현장: 가열시 더 꺼지면 진공보이드, 부풀면 가스포켓. ✅
- **4M**: Machine(보압·보압시간 부족, 멜트온도 과도→냉각시간↑) / Mold(게이트 조기동결, 냉각채널 편재, 금형온도 과도→게이트씰 지연) / Material(결정성 고수축=PP·PA·POM 비결정성의 ~3배) / Method(과도 벽두께).
- **우선순위**: ①보압·보압시간 부족 ②게이트 조기고화(게이트小·금형온도低) ③과도 벽두께(설계) ④결정성 고수축 ⑤냉각 불균일. ✅
- **검증(Gate Seal Study ✅3출처)**: 보압시간 1초씩↑하며 중량 측정→중량 안정(±0.1g)시점=게이트씰. 러너가 먼저 동결되면 설계 재검토.
- **조정**: 보압↑(사출1차압의 50~80%, 과도시 잔류응력), 보압시간↑(게이트씰까지), 멜트온도↓, 게이트확대, coring으로 벽두께 균일화.

### 2.2 Void / Bubble (보이드/기포)
- **식별**: 내부 공동. **진공보이드**(보압부족→스킨 견고→내부함몰, 음압) vs **가스포켓**(용존가스=수분분해·공기, 압해제시 팽창, 양압). 가열 감별 동일. ✅
- **4M**: Machine(보압 부족·조기차단, 배압부족→탈기 안됨, 스크루·역류방지링 마모) / Mold(두꺼운부위 냉각부족, 게이트 위치) / Material(미건조=가스포켓, 결정성 고수축=진공보이드) / Method(V/P 전환 너무 이름).
- **우선순위**: ①결정성+두꺼운벽+보압부족 ②수분 미건조(PA·PC·PBT·PET) ③배압부족(가스포켓) ④게이트 조기고화. ✅
- **검증**: 가열테스트(진공vs가스), 건조강화후 소멸=수분, Gate Seal study, 배압 5~10bar씩↑.
- **조정**: 보압↑, 보압시간↑, 배압 5~10bar 점진↑(가스포켓), 건조 규정준수, coring, 게이트 두꺼운부위 배치.

### 2.3 Warpage (휨/변형)
- **식별**: 이젝트후 형상 이탈. 전체가 한방향 틀어짐=warpage / 균일하게 작거나 큼=shrinkage 문제(구분). ✅
- **메커니즘(근본=불균일 수축)**: 위치별(게이트 근처vs말단) / 두께방향(캐비티vs코어 냉각차) / 배향(유동vs직각) / 두께차(두꺼운곳 느린냉각). ✅
- **결정성 vs 비결정성**: 비결정 수축 0.4~0.8% 등방 / 결정성 1.0~3.0% **직각>유동(결정도메인)**, 냉각속도 민감. ✅
- **GF**: 섬유가 유동방향 수축 억제→**유동 0.3% vs 직각 0.9%(3배)** 이방성 극대화→GF 결정성(PA-GF·PBT-GF) 휨 최악. ✅ 코어-캐비티 온도차 >5~10℃면 두께방향 비대칭 ⚠️.
- **우선순위**: ①GF 결정성+비대칭냉각(최악) ②냉각 불균일(코어-캐비티 온도차) ③보압 불균일 ④결정성 고수축 ⑤두께 불균일.
- **검증**: CMM 정량화, 코어/캐비티 열전대 실측, 냉각회로 유량밸런스, 단면 섬유배향.
- **조정**: 코어-캐비티 온도차 최소화(<5℃ 목표), 금형온도↑(결정성, 잔류응력 완화), 냉각시간↑, 게이트 대칭, 다단 보압, 이젝트직후 냉각지그.

### 2.4 Crack / Crazing / Brittleness (크랙/크레이징/취성)
- **식별**: 이젝터핀 주변=이젝트크랙 / 게이트 방사상=과보압 / 웰드부=후발 / 시간차+용제=ESC. ✅
- **4M**: Machine(과보압→게이트 잔류응력, 사출속도 과다→전단발열→분자량↓) / Mold(이젝터 과대·불균일, 언더컷·드래프트 부족, 냉각불균일→잔류응력) / Material(**과건조→취성**, **미건조→가수분해→분자량↓** PA·PC·PBT·PET, 재생재 과다, 이종오염) / Method(냉각시간 부족, 조립 과하중→ESC).
- **PC ESC ✅**: 잔류응력>15MPa시 ESC 민감 급상승. 금형온도 10~20℃↑→잔류응력 감소. 어닐링 125~135℃×1~4h→잔류응력 60~85%↓, ESC저항 3~5배.
- **우선순위**: ①미건조/가수분해(PA·PC·PBT·PET 최다 단독원인) ②과보압→잔류응력 ③이젝터 불량 ④냉각시간 부족 ⑤재생재·오염.
- **검증**: 편광 응력분석, 수분 측정(Karl Fischer), 보압 단계↓ 테스트, 어닐링 전후.
- **조정**: 건조 규정엄수, 보압↓(과보압시, 1차압 40~60%), 금형온도↑, 이젝터 균일·면적↑, 냉각시간↑, 재생재↓(<20~30%), PC 어닐링.

### 2.5 Dimensional Instability (치수 불안정)
- **식별**: 동일조건 Lot간·캐비티간·시간경과 치수 변동. (warpage=형상, 이건=크기 변동.)
- **4M**: Machine(보압·사출압 변동, 배럴온도 불안정, 역류방지링 마모) / Mold(냉각 불균일, 다캐비티 런너 불균형) / Material(결정성 냉각민감→결정도→수축 변동, **PA 흡습팽창** 수분1%≈치수0.2~0.3%↑, 로트 MFI차) / Method(냉각시간 부족→후수축, 사이클 불일치).
- **후수축 ✅**: 결정성 2차결정화→이젝트후 16~24h 급변, ~1000h까지 미세변화. 고온사용시 두드러짐.
- **우선순위**: ①결정성+냉각/보압 변동 ②PA 흡습팽창 ③냉각시간 부족→후수축 ④기계조건 불안정 ⑤다캐비티 불균형.
- **검증**: 측정타이밍 표준화(이젝트후 24h 안정후), 사이클 중량 모니터(±0.5%), steady state 확인, PA 65%RH 컨디셔닝.
- **조정**: 냉각시간↑, 보압 안정화, 금형온도 안정 유지, 사이클 일정, 어닐링(후수축 선처리), 런너 대칭.

## 3. 표면·재료 불량 (Surface / Material)

### 3.1 Silver Streak / Splay (은선/은줄/스플레이)
- **식별**: 게이트서 유동방향 방사상 은빛 줄무늬. **3종 변별(진단 핵심)**: moisture splay(전면 분산, 건조후 소멸, 수분>임계) / shear splay(게이트 주변·리브, 온도정상인데 지속) / thermal splay(흐름끝·핫러너 다운, 황변·냄새 동반). ✅
- **4M**: Material(흡습수지 PA·PC·PBT·PET·ABS 미건조, 리그라인드 불균일→공기, 오염) / Machine(배럴·핫러너 과열, 스크루RPM·배압 과다→전단열, 서크백 과다→공기역류, 체류>5shot) / Mold(게이트·런너 과소→전단, 블라인드홀 공기트랩, 날카로운 코너) / Method(사출속도 과다→내부공기 유입, 과건조후 방치).
- **우선순위**: ①수분(흡습수지면 건조 먼저) ②열과다(배럴온도·체류) ③전단(게이트·스크루) ④공기트랩(벤팅·서크백). 비흡습(PP·PE)서 은선=즉시 전단/공기 의심. ✅
- **검증(수치 ✅)**: 허용수분 PA66 ≤0.20% / PC·PBT·PET ≤0.02% / ABS ≤0.10%. 이슬점 −29~−40℃. 퍼지후 pyrometer 실멜트온도. 배럴 체류 2~5shot.
- **조정**: (수분)건조시간·온도↑·이슬점 점검 / (열)배럴온도↓·사이클단축·체류↓ / (전단)사출속도↓·게이트확대·배압↓ / (공기)서크백↓·벤팅·리브반경↑.

### 3.2 Flow Mark / Record Groove / Tiger Stripe (흐름자국/레코드홈/타이거스트라이프)
- **식별**: flow mark(유동방향 물결) / record groove(게이트 중심 동심원 홈, 나일론 등 점도민감) / tiger stripe(유동 수직 광택/무광 교대밴드, PP·PP/EPDM/talc). ✅
- **메커니즘**: record groove=차가운 금형서 고화층 형성→벽 직접접촉 불가 / tiger stripe=wall slip(벽면 미끄럼)+결정화, slip-stick 가설 공존(학계 disputed ⚠️).
- **우선순위**: flow mark ①금형온도 낮음 ②사출속도 낮음 ③멜트온도 낮음 / record groove ①게이트 과소 ②금형온도 낮음 ③나일론 과건조 / tiger stripe ①PP 벽면미끄럼(재료고유) ②금형온도 낮음 ③속도변동.
- **조정**: flow mark→멜트·금형온도↑·속도↑·압력↑ / record groove→게이트확대·온도↑·속도↑(나일론 과건조 주의) / tiger stripe→금형온도↑(최효과)·속도균일·고MFR그레이드. ✅High(flow/record), Med(tiger 메커니즘)

### 3.3 Discoloration / Thermal Degradation (변색/열분해)
- **식별**: 전체황변=체류·배럴온도 과다(사이클 재개후 개선) / 국부 흑줄=핫러너 과열·데드존(특정위치 반복) / 흑점=탄화잔류·스크루마모(퍼지후 개선) / 끝단 burn=벤팅 불량 diesel / 색불균일=안료 분산·마스터배치 열분해. ✅
- **4M**: Material(수분+고온→가수분해 PA·PC·PBT·PET, 리그라인드·이종 오염, 금속불순물 Fe·Cu·Ni 촉매) / Machine(배럴·노즐 과열, 체류 과다=shot<배럴20%, 스크루·데드존 탄화, 핫러너 과열) / Mold(벤팅 불량→diesel, 냉각수 막힘→과열) / Method(사이클 정지, 사출속도 과다→전단발열, 퍼지 미흡).
- **우선순위**: ①배럴온도·체류 과다(다운타임 후 빈번) ②수분+고온 가수분해 ③벤팅 diesel(위치고정) ④탄화잔류(퍼지후 개선=확정).
- **검증(수치)**: 배럴 shot 20~80% 용량. 4h+ 정지시 배럴 100℃ 뱅킹. POM 멜트 195~215℃ 엄수(초과→포름알데히드). 퍼지 3~5shot서 소실=잔류탄화.
- **조정**: 멜트온도 5~10℃씩↓, 사이클단축, shot비율 조정, 벤팅 재가공, 퍼지→스크루 점검, 핫러너 프로파일.

### 3.4 Delamination (박리)
- **식별**: fish scale형 층 분리, 손톱으로 층 벗겨짐(은선과 구분=실제 층 분리). 강도 저하 동반→심각도 높음. ✅
- **4M**: Material(**이종수지 오염 1위**=비상용 폴리머 혼입→결합 불가, 용해도 파라미터 차 클수록 심각 / 수분 과다→증기→표층 들뜸 / 리그라인드 오염) / Mold(이형제 과다→표면 오염→계면 저하, 금형온도 과소) / Method(사출속도 과다→전단, 보압시간 부족).
- **우선순위**: ①이종수지 오염(갑자기 발생시 재료교체 이력 확인) ②이형제 과다 ③수분 표층분리 ④금형온도 과소+속도 과다. ✅
- **검증**: 손톱 긁기, 수지 로트 이력, DSC 용융점(이종 피크), 이형제 사용량 로그.
- **조정**: 완전 퍼지·원료검사·리그라인드 배제, 이형제 최소화, 건조 점검, 금형온도↑·속도↓·보압시간↑.

### 3.5 Fiber Read-out / 표면 백화 (GF 섬유 부상)
- **식별**: GF수지 표면 백색 방사상 흔적·거침, 흑색 부품서 두드러짐. ✅
- **메커니즘(fountain flow)**: 섬유(밀도 2.5)·수지 밀도차+점도구배로 섬유 흐름방향 배향, 금형온도 낮으면 표면수지 급냉→스킨층 형성 전 섬유 노출 고화. 배럴온도↑→유동성↑→분산↑ / 금형온도↑→냉각↓→섬유 피복 시간 확보. ✅
- **4M**: Material(GF함량 과다 >30~40%, sizing 불량, 점도부족) / Machine(스크루RPM 과다→섬유파손, 배럴온도 과소) / Mold(**금형온도 과소=핵심**, 게이트 과소→전단·파손) / Method(사출속도 과소→조기냉각, 단 과다→jet).
- **우선순위**: ①**금형온도 과소(단일 최대 개선)** ②배럴온도 부적절 ③스크루RPM 과다→파손 ④사출속도 과소.
- **검증(수치 ⚠️Med-High)**: PA66+33%GF 금형 110℃서 부상 억제 사례 / 배럴 275~280℃(일반 PA66+10~30℃). 스크루RPM 비강화 대비 낮게.
- **조정**: 금형온도↑(최효과), 배럴온도 비강화 대비 10~30℃↑, 사출압↑, 속도↑(jet주의), 스크루RPM↓, 게이트확대.

### 3.6 Surface Roughness / Gloss Variation (표면거침/광택 불균일)
- **식별**: 광택 전이(유동 수직 고/저광택 교대 경계선) vs 전체 거침. flow mark(줄무늬 패턴)와 구분=광택 수치차가 주증상. ✅(MDPI 2020)
- **메커니즘**: fountain flow가 표면층 지배. 금형 복제(압력→스킨 밀착→전사→고광택) vs 수축·점탄성 경화(저광택). 금형온도↑→냉각↓→복제율↑→광택↑. gloss transition=금형온도 불균일+유동전면 속도변동.
- **우선순위**: ①금형온도(과소·불균일, 단일 최대) ②유동전면 속도변동 ③보압 과소(복제 불충분) ④금형 표면상태.
- **검증**: 광택계 60°/20° GU 매핑, 금형 표면온도 편차(±5℃ 권장), V/P후 압력변동.
- **조정**: 금형온도↑, 보압↑(복제율), 사출속도 균일(다단), 냉각채널 점검, 폴리싱, 결정성=금형온도 Tg 이상(고광택시). MDPI: 보압 500bar서 gloss transition 소멸.

## 5. 크로스커팅 레이어 (전 불량에 걸치는 변조 요소) ★

### 5.1 결정성(semi-crystalline) vs 비결정성(amorphous)
| | 결정성 | 비결정성 |
|---|---|---|
| 수축률 | 1.0~2.5% | 0.4~0.8% |
| 방향성 | 비등방(유동≠직각) | 등방 |
| 수축 원인 | 열수축+결정화수축 | 열수축만 |
| 금형온도 의존 | 높음(냉각속도→결정화도) | 낮음 |
- 결정성: 금형 내 결정화 미완→탈형후 후수축. 권장 금형온도(Tc 근처) 엄수. ✅
- 결정성 예: PA·PP·POM·PBT·PET·PPS·LCP·PEEK / 비결정: PC·ABS·PMMA·PS·PSU·PEI.

### 5.2 GF(유리섬유) 강화
- 섬유 유동방향 정렬→수축 유동<직각→차동수축→휨. PA6-GF 유동 0.3% vs 직각 0.9%(3배). ✅
- 배향이 처리조건보다 휨 지배→게이트 위치·CAE 중요. core-skin 구조(표면=유동정렬, 코어=무작위).
- 표면 백화=금형온도 낮아 섬유 캡슐화 실패 (3.5 참조).

### 5.3 Super Engineering Plastics 금형온도 의존 ✅
| 수지 | 금형온도 | 근거 |
|---|---|---|
| PEEK | **170℃+** (tool surface) | Victrex 공식: "표준 결정도 위해 최소 170℃ 도달 필수" |
| PEEK-HT/ST | 190 / 200℃+ | Victrex 공식 |
| PPS | 120~160℃ | 3출처 일치 |
| PPA(고열 HH) | >80℃, **오일 TCU 필수** | Solvay 공식 |
| LCP | 65~150℃ (하한 65 엄수) | 2출처 ⚠️ |
- 원리: Tg 높음(PPS~90, PEEK~143, PPA~130). 금형온도 Tg 이하→결정화 급감→비정질→강도·치수·웰드 취약. LCP는 결정화 매우 빨라 충전 전 결정화→미충전 위험, 하한 엄수.

### 5.4 금형온도 셋팅값 vs 실측값 갭 ★★ (진우 현장통찰 = 학술 검증)
- **Victrex 공식 직접 인용**: "오일 히팅은 배관 손실로 히터 설정온도가 실제 금형표면 온도보다 상당히 높을 수 있다. **40℃ 손실이 흔하다(Heat losses as high as 40°C are common).**" ✅
- 원인: 배관 열손실(최대 40℃), 금형 개폐시 플래튼 heat sink, 열평형 도달(소변경 20분·대변경 수시간), 웜업샷(10샷·15분+).
- **가열매체 온도 상한**: 수냉 TCU 121~149℃ / 가압수 160℃ / 오일 149~288℃ / 전기 카트리지(상한 높으나 균일성 낮음·hot spot). ✅ ~149℃ 이상은 steam flashing→오일 TCU 필수.
- **실용 규칙**: 오일 TCU는 목표보다 설정 20~40℃ 높게. 대형금형일수록 갭 큼. 금형-플래튼 단열판(34% 에너지 절감). 실측은 표면 2mm내 열전대, 복수 위치.
- **앱 적용(거버넌스)**: 금형 실측온도 입력 강제 X(현장 센서 부재). 셋팅값으로 판단하되 super EP/GF+금형온도 의존 불량시 **출력 가이드로 조언**: "셋팅≠실제, 개폐로 식음, GF→백화·super EP→결정화부족, 초기샷만=워밍업·지속=가열부족, 외부 TCU 한계시 카트리지(히터봉) 검토." 가열방식(온수기/온유기/카트리지)은 선택 입력.

### 5.5 Scientific Molding 방법론
- **Decoupled II**: 충전 95~99%(속도제어)+팩킹/홀딩 통합 / **III**: 충전 80~85%+2차충전(보압)+홀딩, 캐비티 압력센서 폐루프. 충전=속도제어, 팩/홀드=압력제어(분리=Decoupled). ✅
- **Gate Seal Study**: 보압시간 단계↑→중량 안정시점=씰, +1~2초 여유. 금형온도·게이트·배압·홀드압·멜트온도 변경시 재실시. ✅
- **4M**: Man/Machine/Material/Mold. DOE 결합→영향도 정량화→Process Window.
