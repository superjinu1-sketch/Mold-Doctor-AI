# Defect Taxonomy — 단일 진실 소스 (v1.0)

> **버전 v1.0 (2026-06-05).** 코드(`defect-kb.ts`)와 충돌 시 이 문서 우선.
> 상세 근거·출처·메커니즘 = `docs/defect-research-synthesis.md`. 이 문서는 진단이 따르는 **규범**.
> 거버넌스: 출력은 "추정/조정안"(진단 단어 금지), 100% 정확 주장 금지, 신뢰도 명시, 제조사 브랜드 중립.

## 개정 가이드 (출시 후 수정·추가)
- **불량 추가**: 아래 목록에 항목 추가 → defect-kb.ts에 DefectNode 1개 추가(기존 노드 안 건드림) → KB_VERSION bump → eval.
- **원인/우선순위 보정**: 해당 불량 분기의 기본확률·순위만 수정. 현장 피드백(플라이휠) 누적 시 이 경로로.
- **공유 게이트 수정**: 한 곳(게이트 정의)만 고치면 참조하는 모든 불량에 반영(중복 수정 금지).
- 수치 범위(건조·멜트·금형)는 여기 박지 말고 `resin-kb.ts` 참조(중복 0). taxonomy는 분기 로직만.

## 1. 분류 체계 (3축)
- **공정 Phase**: 재료준비 → 충전 → 보압 → 냉각 → 이형
- **결과 유형**: 표면 / 치수·형상 / 내부·구조 (보조)
- **원인 4M**: Machine / Material / Mold / Method
- 원칙: 대부분 복합 원인 → 트리는 Primary(1순위)와 Contributing(기여) 구분.
- **텍스처 변별(필수)**: 표면 백색·뿌연 얼룩/가루가 "닦으면 옅어지거나 지워지면" = 표면 부착물(금형 석출 plate-out·가스 응축·이형제 전사). 닦아도 안 지워지면 = 구조적(웰드라인·플로우마크·지속형 은선·표면 요철). 이 한 줄이 외관 백색 불량의 1차 분기를 가른다.

## 3. 불량별 규범 (정의 / 1순위 원인 / 주 조정) — 상세는 synthesis.md
각 불량은 defect-kb.ts에서 {식별포인트, 분기[순위·4M·기본확률·활성조건·증거·검증·조정], 패턴힌트, 공유게이트참조, 우선순위로직} 구조. 아래는 규범 요약.

- **Short Shot**: 유동에너지<저항. 1순위 사출압·속도·멜트온도 부족 / 특정캐비티=게이트밸런스. 조정 압·속·온도↑·벤트.
- **Flash**: 캐비티압>클램프력. 1순위 클램프력 부족+과압 / 특정위치=파팅면 마모. 조정 클램프↑·사출압↓·V/P 앞당김.
- **Jetting**: 관성>점성 자유제트. 1순위 게이트(위치·크기)+속도과고. 조정 게이트통과 속도↓(다단)·멜트온도↑·게이트확대.
- **Weld Line**: 선단 재확산 불충분. 1순위 멜트온도 부족(강도 71% 좌우)/게이트위치(구조). 조정 멜트·금형온도↑·홀드압↑·위치이동.
- **Air Trap/Burn**: 갇힌공기 단열압축 발화. 1순위 벤트 부족/막힘 / 클램프 과고→벤트압착 주의. 조정 벤트 추가·청소·속도↓.
- **Flow Mark**: 선단 점진 고화. 1순위 금형온도 낮음·속도 낮음. 조정 금형온도·속도·멜트온도↑.
- **Sink Mark**: 보압 보상 부족. 1순위 보압·보압시간 부족 / 게이트 조기동결. 조정 보압↑(게이트씰까지)·게이트확대·coring.
- **Void/Bubble**: 진공보이드(보압부족) vs 가스포켓(수분·배압부족). 1순위 결정성+두꺼운벽+보압부족 / 수분. 조정 보압↑·배압↑·건조.
- **Warpage**: 불균일 수축. 1순위 GF결정성+비대칭냉각 / 냉각 불균일. 조정 코어-캐비티 온도차<5℃·냉각시간↑·대칭게이트.
- **Crack/Crazing**: 1순위 미건조/가수분해(PA·PC·PBT·PET) / 과보압 잔류응력. 조정 건조 엄수·보압↓·금형온도↑·PC 어닐링.
- **Silver Streak**: 3종 변별 — moisture(건조후 소멸)/shear(게이트주변 지속)/thermal(황변동반). 1순위 수분(흡습수지). 조정 (수분)건조↑ /(전단)속도·배압↓ /(열)배럴온도↓. **금형온도 게이트 미적용.**
- **Discoloration**: 1순위 배럴온도·체류 과다 / 수분 가수분해 / 벤팅 diesel(위치고정). 조정 멜트온도↓·사이클단축·퍼지.
- **Delamination**: 1순위 이종수지 오염 / 이형제 과다. 조정 완전퍼지·원료검사·이형제 최소.
- **Fiber Read-out(GF백화)**: 1순위 금형온도 과소(섬유 캡슐화 실패). 조정 금형온도↑·배럴온도↑·스크루RPM↓.
- **Mold Deposit/Plate-out**: 닦으면 옅어지는 표면 부착물(백색·뿌연 가루·얼룩). 1순위 수지 휘발분·첨가제 석출(고멜트·장체류) / 벤트 가스응축 / 이형제 전사. 조정 멜트온도↓·체류단축·금형 표면 세정·벤트 청소. **변별 핵심: 닦임=부착물 → 웰드/플로우/은선 배제.**
- **Surface/Gloss**: 1순위 금형온도(과소·불균일) / 유동전면 속도변동. 조정 금형온도↑·보압↑·속도 균일(다단).
- (16~29 상세는 synthesis.md 4·기타 표 참조)

## 4. 공유 게이트 (조건부, 여러 불량 참조)

### 4.1 금형온도 부족 게이트 (mold_temp_insufficient)
- **적용 불량**: Weld Line·Flow Mark·Short Shot(2~3순위)·표면결정화/Gloss·Fiber Read-out·Tiger Stripe·Record Groove·Gate Blush. (조건부: Warpage)
- **트리거**: 위 불량 + (셋팅 금형온도 < resin-kb 권장 OR super EP OR GF강화). **셋팅값만으로 판단, 실측 입력 요구 X.**
- **출력 가이드(조언)**: 셋팅≠실제 금형온도(개폐로 식음, 큰 금형일수록·Victrex 오일배관 40℃ 손실) / 안 올라가면 GF→표면백화·super EP→결정화부족·광택저하 / 초기샷만=워밍업 부족(예열샷↑)·지속=가열 근본부족 / 외부 TCU(온수기121~149℃·가압수160℃·오일149~288℃) 한계시 카트리지 히터(히터봉) 직접삽입 검토 / 금형 실측 권장(강제 X).
- **입력**: 가열방식(온수기/온유기/카트리지) 선택 입력.
- **미적용(false positive 방지)**: 은선·플래시·버닝·싱크 등.

## 5. 메타 분기 (기존 유지)
- **No_Defect_Detected**: 이미지 선명·성형품이나 불량 패턴 없음. confidence=low, causes/adjustments=[].
- **Image_Unreadable**: 단색·흐림·과노출·사출 무관. confidence=low, 재촬영 안내.
- **Insufficient_Input**: 불량유형 미선택+셋팅 거의 없음. 추가 정보 요청.
