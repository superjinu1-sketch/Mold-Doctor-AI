# Defect Taxonomy — 단일 진실 소스

코드와 충돌 시 이 문서 우선.

## 1. Flash (플래시, 버)
- 정의: 파팅라인/에어벤트로 수지가 새어나와 굳은 얇은 막
- 주 원인: 클램프력 부족, 과보압, 파팅라인 마모, 수지 점도 과다 저하 (과온)
- 조정 방향: 클램프력 ↑, 보압 ↓, 배럴 온도 ↓, 에어벤트 점검

## 2. Short Shot (미성형)
- 정의: 캐비티 말단까지 수지가 도달하지 못한 상태
- 주 원인: 사출속도/압력 부족, 배럴 온도 낮음, 게이트 막힘, 벤트 부족
- 조정 방향: 사출속도 ↑, 보압 ↑, 배럴 온도 ↑, 벤트 확보

## 3. Sink Mark (수축자국)
- 정의: 두꺼운 부위 표면이 내부 수축으로 함몰
- 주 원인: 보압 부족, 냉각 시간 부족, 게이트 크기 부족, 리브 설계 문제
- 조정 방향: 보압 ↑, 보압 시간 ↑, 냉각 시간 ↑, 게이트 확대

## 4. Weld Line (웰드라인)
- 정의: 유동 선단 두 흐름이 합류한 선
- 주 원인: 배럴/금형 온도 낮음, 사출속도 낮음, 벤트 부족
- 조정 방향: 금형 온도 ↑, 사출속도 ↑, 벤트 확보, 게이트 위치 재검토

## 5. Burn Mark (탄화)
- 정의: 갇힌 공기가 단열 압축으로 수지를 태운 흑갈색 자국
- 주 원인: 벤트 부족/막힘, 사출속도 과다, 배럴 온도 과다
- 조정 방향: 벤트 확보, 사출속도 ↓ (말단 감속), 배럴 온도 ↓

## 6. Flow Mark (흐름자국)
- 정의: 표면에 물결/링 형태 무늬
- 주 원인: 금형 온도 낮음, 사출속도 낮음, 수지 점도 불균일
- 조정 방향: 금형 온도 ↑, 사출속도 ↑ (초기), 건조 상태 점검

## 7. Jetting (제팅)
- 정의: 게이트에서 튀어나온 수지가 구불구불 자국 남김
- 주 원인: 사출속도 과다(초기), 게이트 크기 부족, 게이트 위치 불량
- 조정 방향: 초기 사출속도 ↓, 다단 속도 프로필, 게이트 확대/재배치

## 8. Silver Streak (은선)
- 정의: 표면 은빛 줄무늬
- 주 원인: 수분 (건조 부족), 휘발성 분해, 과온, 전단 발열
- 조정 방향: 건조 강화, 배럴 온도 ↓, 스크류 회전수 ↓, 역압 조정

## 9. Warpage (변형)
- 정의: 이형 후 치수/형태 변형
- 주 원인: 불균일 냉각, 잔류 응력, 섬유 배향, 이형 시점 미성숙
- 조정 방향: 금형 냉각 균일화, 냉각 시간 ↑, 보압 프로필 조정

## 10. No_Defect_Detected (이상 없음)
- 정의: 이미지 판독은 됐으나 위 9종 불량 형상이 식별되지 않은 상태
- 사용 조건: 이미지가 선명하고 성형품이 식별되나 표면·형상에 불량 패턴이 없을 때
- 사용자 안내: "이 이미지에서 불량 형상이 검출되지 않았습니다. 의심되는 부위를 확대 촬영하거나 다른 각도 사진을 추가하세요."
- 출력 규칙: confidence = "low" / causes = [] / adjustments = []

## 11. Image_Unreadable (판독 불가)
- 정의: 이미지가 단색·흐림·과노출·사출 부품 무관 등으로 불량 형상 식별 자체가 불가한 상태
- 사용 조건: 단색 이미지, 극도로 흐린 사진, 음식·자연·문서 등 사출 무관 사진, 극저해상도
- 사용자 안내: "이미지를 판독할 수 없습니다. 불량 부위를 정면으로 충분한 조명 아래 촬영한 사진을 재업로드하세요."
- 출력 규칙: confidence = "low" / causes = [] / adjustments = []
